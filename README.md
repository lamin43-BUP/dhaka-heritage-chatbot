The Dhaka Heritage Chatbot is an interactive AI-powered chatbot built using Python, TensorFlow, and NLTK. It helps users explore and learn 
about the historical and cultural landmarks of Dhaka, Bangladesh. The chatbot understands natural language queries, identifies the intent 
using a trained neural network, and provides accurate responses about Dhaka's rich heritage. It is designed to be user-friendly and can be 
run easily in Google Colab or a local Python environment. Whether you are a student, tourist, or history enthusiast, this chatbot provides 
a fun and educational way to discover Dhakaâ€™s iconic sites.

<br>
<br>

## Features:
* Recognizes user questions using Natural Language Processing (NLP).
* Predicts intents from user input using Bag-of-Words and a Neural Network.
* Provides information on historical sites of Dhaka like Lalbagh Fort, Ahsan Manzil, Curzon Hall, Dhakeshwari Temple, Armenian Church of
  Dhaka, Old Dhaka City Walls and Shahid Minar. 
* Interactive chat interface in Google Colab or local Python environment.

<br>

## Requirements
* Python 3.x
* TensorFlow
* NLTK
* NumPy

Install the required packages using pip:
```bash

```
